Developer Console Guide

This guide details how to access and use the in-game developer console for testing and debugging purposes.

Accessing the Console

To open the developer console overlay, press the following key combination anywhere in the game:

Ctrl + Shift + ~ (Tilde/Backtick)

Authentication

Upon opening the console for the first time, you will be prompted for a password.

Password: voids-arcana-dev

Command Reference

Once authenticated, you can type the following commands into the input field at the bottom of the console window.

Command Usage Example Description
help help Lists all available commands.
set_name set_name Hero Changes the character's display name.
set_dob set_dob 1990-01-01 Sets the character's Date of Birth.
set_element set_element fire Sets DOB to a date matching the element (fire, earth, air, water).
set_gender set_gender Female Sets the character's gender.
set_trait set_trait E 15 Sets a specific personality trait value (N, E, O, C, A).
set_class set_class mage Changes the character's class ID.
reset_stats reset_stats Resets all allocated stat points and refunds them based on level.
reset_skills reset_skills Clears all learned skills.
add_currency add_currency gold 5000 Adds the specified amount of currency (gold, silver, diamonds).
reset_daily reset_daily Resets the daily quest limit, allowing you to take another daily quest immediately.
reset_cd reset_cd Resets all skill cooldowns.
reset_quest_cd reset_quest_cd Resets cooldowns for all completed quests (sets completion time to long ago).
restore_all restore_all Fully restores HP, SP, and Stamina to their maximum values.
complete_quest complete_quest Instantly completes the currently active quest successfully.
download_logs download_logs Downloads the current session's console logs as a .txt file.

Tips

  • Use the Up and Down arrow keys to cycle through your command history.
  • Commands are case-insensitive (e.g., SET_NAME works the same as set_name).
  • If a command fails, check the console output for error messages.
